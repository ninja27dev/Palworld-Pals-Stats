# 📊 Palworld Data Visualization Dashboard

An **interactive data visualization dashboard** built using **Python, Dash, Plotly, and Pandas** to analyze the characteristics of **Palworld creatures (Pals)**.

This project explores different attributes such as **types, rarity, combat stats, skills, speed, price, and item drops** through interactive visualizations.

---

## 🚀 Project Overview

The dashboard analyzes the **Palworld dataset** and provides insights using various interactive charts and graphs.  
It helps users understand relationships between different Pal attributes and compare their abilities.

Key analysis areas include:

- Pal distribution by **type**
- Pal distribution by **rarity**
- **Combat statistics analysis**
- **Work skills and suitability**
- **Speed attributes comparison**
- **Economic value and price distribution**
- **Item drops from Pals**

---

## ✨ Features

### 1️⃣ Pal Distribution and Characteristics
- Count of Pals by **Type**
- Count of Pals by **Rarity**
- Average combat statistics by type
- **Price vs Rarity scatter plot**
- Distribution of stats using **histograms and box plots**

---

### 2️⃣ Pal Skills and Work Suitability
- Stacked bar chart of **skill levels by Pal type**
- Distribution of **different work skill combinations**
- **Skill correlation heatmap**
- **Parallel coordinates plot** to compare multiple skills

---

### 3️⃣ Pal Combat Attributes
- **3D combat stats visualization**
- Radar chart for comparing combat abilities
- Relationship between **Attack and Defense**
- Speed analysis:
  - Walk Speed
  - Run Speed
  - Ride Sprint Speed

---

### 4️⃣ Pal Economy and Value
- **Price distribution by rarity**
- **Violin plot for price analysis**
- Visualization of **Top 20 most common drops**
